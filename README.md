# Rule-Based AI Chatbot

## Project Overview

This project was developed as **Project 1** during my **Artificial Intelligence Internship at DecodeLabs**. The objective was to build a simple Rule-Based AI Chatbot using Python that responds to predefined user inputs through conditional statements and decision-making logic.

Unlike modern AI chatbots that rely on Machine Learning or Natural Language Processing (NLP), this chatbot uses basic Python programming concepts such as `if-elif-else` statements, loops, and string manipulation to simulate conversations.

## Features

- Interactive command-line chatbot
- Continuous conversation using a `while` loop
- Rule-based responses using `if-elif-else`
- Handles greetings (Hi, Hello, Hey)
- Responds to common questions
- Displays a help menu
- Handles unknown inputs gracefully
- Exits using `bye`, `exit`, or `quit`
- User input sanitization using `strip()` and `lower()`

## Technologies Used

- Python 3
- Conditional Statements
- While Loop
- String Manipulation
- Control Flow

## Learning Outcomes

Through this project, I learned how rule-based systems work, how to process user input, implement decision-making logic, create interactive console applications, and organize Python code for better readability. This project also strengthened my understanding of the fundamentals of Artificial Intelligence and conversational systems.

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python chatbot.py
```

4. Start chatting with the bot.
5. Type `bye` to exit the program.
